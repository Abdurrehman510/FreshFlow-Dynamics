# 📖 User Guide: FreshFlow Dynamics

This guide explains how to use the different sections of the FreshFlow platform.

## 👤 For Business Owners (Admin Dashboard)

### 📊 Monitoring the Dashboard
The **Overview** tab displays real-time KPIs:
- **Total Wastage**: The total value lost due to product expiry.
- **Critical Stock**: Products expiring within 24–48 hours that need immediate attention.
- **Customer Volume**: Active users registered on the platform.

### 📝 Inventory Management
Navigate to the **Products** section to:
- **Add Product**: Input the category and expiry date. The engine will automatically set a price.
- **Reprice Now**: Force a platform-wide repricing check without waiting for the nightly job.

### 📉 Demand Forecasting
The **Forecast** section uses AI to tell you what to order:
- **Reorder Alerts**: Highlight products that are projected to run out within 7 days.
- **Demand Trends**: Shows if a product's popularity is rising or falling.

---

## 🛍️ For Customers (Customer Portal)

### ✨ Finding the Best Deals
The **"Flash Deals"** section automatically highlights products with high expiry-based discounts. 
- Items expiring tomorrow might be listed at **50% off**, perfect for immediate consumption.

### 💳 Using your Wallet
- Each user has a virtual wallet.
- Orders can be paid for using **Wallet Balance** or **Cash on Delivery**.
- Wallet transactions are instantaneous and recorded in your order history.

### 📦 Tracking Orders
- View your **Order History** to see past purchases and current order status (Pending → Confirmed → Delivered).
